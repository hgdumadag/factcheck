Key Components Explained
========================

Backend (Python/FastAPI)
------------------------

1. app.py
   - FastAPI application setup
   - CORS configuration
   - API endpoints:
     * GET  / (health check)
     * POST /api/factcheck/text
     * POST /api/factcheck/url
     * POST /api/factcheck/image

2. modules/qwen_client.py
   - Wrapper for Alibaba Qwen 3 LLM
   - Methods:
     * chat() - General chat completion
     * extract_json_response() - Get structured JSON
     * simple_prompt() - Quick prompts

3. modules/input_processor.py
   - Processes three input types:
     * Text: Direct processing
     * URL: Web scraping (newspaper3k + BeautifulSoup)
     * Image: OCR with Tesseract

4. modules/claim_extractor.py
   - Uses Qwen to extract:
     * Main claim
     * Key facts (checkable/non-checkable)
     * Named entities
     * Dates mentioned

5. modules/search_engine.py
   - Multi-source search:
     * DuckDuckGo (free, no API key)
     * Google Custom Search (optional)
     * Fact-check site search
   - Returns structured evidence

6. modules/context_analyzer.py
   - **KEY INNOVATION**
   - Identifies missing context
   - Generates full picture summary
   - Extracts timeline from evidence
   - Uses Qwen for analysis

7. modules/verifier.py
   - Calculates confidence scores:
     * Source agreement
     * Source quality
     * Context completeness
     * Fact-check coverage
   - Determines final verdict


Frontend (React)
----------------

1. App.js
   - Main application component
   - Sub-components:
     * VerdictCard - Shows verdict & confidence
     * MissingContextCard - Context alert (KEY FEATURE)
     * TimelineView - Event timeline
     * SourcesList - Evidence sources
     * ScoresCard - Detailed metrics

2. App.css
   - Modern gradient design
   - Responsive layout
   - Color-coded verdicts
   - Smooth animations

Environment Variables
====================

Required:
- DASHSCOPE_API_KEY: Alibaba Qwen API key

Optional:
- GOOGLE_API_KEY: Google Custom Search API key
- GOOGLE_CSE_ID: Google Custom Search Engine ID
- API_HOST: Backend host (default: 0.0.0.0)
- API_PORT: Backend port (default: 8000)
