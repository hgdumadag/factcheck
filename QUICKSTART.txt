ğŸ”§ TROUBLESHOOTING
==================

Backend Won't Start
-------------------
- Activate venv: venv\Scripts\activate
- Install deps: pip install -r requirements.txt
- Check .env file exists with API key

Frontend Won't Start
--------------------
cd frontend
npm install
npm start

API Errors
----------
- Verify DASHSCOPE_API_KEY in .env
- Check API key is active on dashscope.console.aliyun.com
- Ensure you have API credits

Port Conflicts
--------------
Backend uses 8000, frontend uses 3000
Change in .env if needed:
    API_PORT=8001
